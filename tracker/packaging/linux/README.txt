ChipNomad Linux Package
=======================

This is a portable Linux build of ChipNomad, a chiptune music tracker.

REQUIREMENTS:
- SDL2 library (install with: sudo apt install libsdl2-2.0-0)
- x86_64 Linux system

INSTALLATION:
1. Extract this archive to any directory
2. Make sure SDL2 is installed (see requirements above)
3. Run ChipNomad using one of these methods:
   - Double-click chipnomad.sh (if your file manager supports it)
   - Run ./chipnomad.sh from terminal
   - Run ./chipnomad directly from terminal

CONTENTS:
- chipnomad: Main executable
- chipnomad.sh: Launcher script
- instruments/: Sample instruments
- pitch-tables/: Pitch table files
- projects/: Demo projects
- chipnomad-icon.png: Application icon

For more information, visit: https://chipnomad.org
Join our Discord: https://discord.gg/PJarAn2QCW

This build works on most Linux distributions including Ubuntu, Debian, 
Fedora, Arch Linux, and Steam Deck (Desktop Mode).
